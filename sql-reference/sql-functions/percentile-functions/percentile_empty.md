# percentile_empty

## 功能

构造一个 `percentile` 类型数值

## 语法

```Haskell
PERCENTILE_EMPTY();
```

## 参数说明

无

## 返回值说明

返回值的数据类型为 PERCENTILE

## 示例

```Plain Text
mysql> select PERCENTILE_EMPTY();
+--------------------+
| percentile_empty() |
+--------------------+
| NULL               |
+--------------------+
1 row in set (0.12 sec)
```
